About Lingenia

This is a Conlang generator for creating phonology and indivdiual words. You can choose the number of vowels and consonants that you want your language to have (or let the program pick them for you). The program will then generate the phoneme components of the language, based on the probability of each phoneme occuring in a real world language. You can then use these generate a number of words.

Alternatively, if you want to specify your own phonology, you can click on each sound you want to include and it will be added to the phonology list. Double clicking on a sound will play it so you can hear what it sounds like.
